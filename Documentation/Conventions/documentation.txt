_____________________Oasis__Conventions__Documentation_________________________

Here's the list of all conventions used in this operating system source tree:
	- All folders are named with the first capital letter, all file names start with the
		lowercase letter.
	
	- Functions are called like this: _printText

	- Variables are called like this: someVariable

	- Namespaces are called like this: SomeNamespace

	- Comments are written like this:
		;
		;	Comment
		;
